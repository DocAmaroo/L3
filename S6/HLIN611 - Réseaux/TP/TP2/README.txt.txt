Execution :

> Ouvrir 2 terminals dans le dossier bin
> Dans le premier executer : ./serveur <PORT>
> Dans le second executer : ./client <SERVER ADRESS> <PORT>

<SERVER ADRESS> : je vous conseil l'adresse de votre localhost : 127.0.0.1, à vrai dire j'ai pas testé d'autre adresse ^^
<PORT> : Il faut prendre un nombre > 1023

> Suivez les instructions et tout est niquel ;)
